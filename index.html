Digital Collections UI Task

Introduction

This page demonstrates three types of views using a set of collection data retrieved from JSON with respective supported fields listed below:

  • Gallery View
    • Title
    • Image
    • Date
    • Description
    • Item Type
  • Full-width Detailed View
    • Title
    • Image
    • Date
    • Description
    • Language
  • Featured Item View
    • Title
    • Image
    • Date

Tools/Frameworks/Processes

  • CSS Flexbox
  • Bootstrap
  • jQuery
  • Mobile Responsiveness

Some Notes

With more time, accessibility features should also be implemented.

Buttons are for demo only and do not link anywhere.

The first two views generated the dataset provided, whereas the last one is demoing one item only.
